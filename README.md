# Fanuc FTP Client GUI

A desktop GUI application for transferring TP program files to Fanuc robots via FTP.

Build with Python and Tkinter

## Features
-Select robot by name or enter a custom IP address
-Connect to robot controller via FTP
-Browse files on the robot directory
-Upload '.tp' program files to selected destination folders

## Requirements
-Python 3.x
-tkinter(included in standard Python installation)

## How to Run
'''bash
pyrhon FanucRobotFTP.py
'''

## Project Status
GUI is functional. FTP communication is under development

## Notes
This repository contains the GUI only.
The FTP communication library is not included.
